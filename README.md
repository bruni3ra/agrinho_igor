# Agro Forte e Futuro Sustentável

Este é um site institucional simples em HTML, CSS e JavaScript criado para apresentar uma proposta de comunicação visual para um projeto do agro focado em sustentabilidade, produção responsável e consumo equilibrado.

## O que o projeto faz

- Exibe uma página única com seções de apresentação, valores, ações e acesso rápido.
- Apresenta cards e blocos de conteúdo que explicam temas como produção sustentável, consumo equilibrado e inovação.
- Inclui um cabeçalho com navegação, chamadas para ação, seções de iniciativa e um rodapé.
- Tem um botão de voltar ao topo que melhora a navegação em páginas longas.
- Funciona como um protótipo visual de página de destino para iniciativas agrícolas verdes.

## Principais arquivos

- `index.html` — estrutura de conteúdo da página.
- `style.css` — estilos visuais, layout responsivo e suporte a animações básicas.
- `script.js` — comportamentos interativos da página (por exemplo, scroll top e ano dinâmico).

## Estrutura do site

- `nav.top-nav` — navegação principal com links internos e contato por e-mail.
- `header.hero` — seção hero com título, descrição e botões de chamada para ação.
- `section#valores` — cards que destacam valores centrais do projeto.
- `section.initiative-grid` — blocos de iniciativas com ilustrações e descrições.
- `section#acesso` — acessos rápidos para explorar ações, aprender sobre sustentabilidade e falar com a equipe.
- `section.highlight` — mensagem de destaque para reforçar a proposta de valor.
- `section#acoes` — lista de práticas e uma citação inspiradora.
- `footer.footer` — rodapé com informações de marca.

## Como usar

1. Abra o `index.html` no navegador.
2. Navegue pelas seções usando os links no menu.
3. Use o botão de voltar ao topo para retornar rapidamente ao início da página.

## Objetivo do projeto

O objetivo é demonstrar um layout moderno e responsivo com foco em conteúdo sustentável, usando tecnologias básicas de front-end. Ele serve como base para um site institucional de projeto rural, cooperativa ou iniciativa agroecológica.
